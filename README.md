A beautiful, physics-driven progress indicator / loader with **swarm snakes** that feel alive.   
   
- **Snakes** move along track (circular)    
- **Rebel Snake** (orange, moves backwards)   
- **Glow + Gradient-Tails** + Blur   
- **Entrance/Exit animations** (Fade, Grow, Implode, Explode, BlackHole)  
- **Indeterminate** (endless loop) or **Determinate** (real progress)
